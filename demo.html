✨ Features Implemented:

  • Online Status: Shows "Online" in green when user is active
  • Offline Status: Shows "last seen [time ago]" in gray when user is offline
  • Precise Time Formatting:
    • < 30 seconds: "just now"
    • 30-60 seconds: "less than a minute ago"
    • 1-59 minutes: "X minutes ago"
    • 1-23 hours: "X hours ago"
    • 24+ hours: "X days ago"
  • Real-time Updates: Status updates automatically every 30 seconds
  • Right-side Display: User name and status shown on the right side of header
